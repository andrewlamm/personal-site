# personal site

my [personal site](https://andrewl.dev/)

### how to run

1. `git clone git@github.com:superandybean/personal-site.git`
2. `cd personal-site`
3. `npm install`
4. `gatsby develop`
5. visit site on `localhost:8000`

### to deploy

make sure you are logged in with `firebase login`
`firebase deploy`
